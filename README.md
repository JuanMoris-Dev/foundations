Termos e explicações

.gitignore -> Ignora Arquivos dos quais você escolhe, ele não commita no repositorio remoto nem versiona no repositorio local.

Issues (Pedido de ajuda) -> Descobriu um problema, não sei como resolver e precisa de ajuda para resolver.

Pull Request (Ajuda oferecendo a solução do problema)-> Encontrou um problema, Analisei o código, Aprendi como resolver, mexi no código e dei uma solução do problema.
